# Task 00003: Implement store.py — FileStore with fcntl locking and 2-level directory sharding

## Role
data-engineer

## Objective
Implement `src/corpus_council/core/store.py` containing the `FileStore` class that encapsulates all user data file I/O with fcntl locking, 2-level directory sharding, atomic JSON writes, and safe JSONL appends. The class must pass `mypy --strict` and `ruff check`. No other module in the codebase should use raw `open()` for user data writes.

## Context
Task 00001 created the package scaffold. Task 00002 created `config.py` with `AppConfig`. The `FileStore` is the single gateway for all user/session persistence. It must implement the exact sharding and locking behavior specified in `project.md` and `data-engineer.md`.

User data path structure:
```
data/users/{user_id[0:2]}/{user_id[2:4]}/{user_id}/
  chat/
    messages.jsonl
    context.json
  collection/
    {session_id}/
      session.json
      messages.jsonl
      collected.json
      context.json
```

Path sharding: `user_id = "abc123ef"` → `data/users/ab/c1/abc123ef/`. A `user_id` shorter than 4 characters must raise `ValueError`.

JSONL append schema for each turn (`chat/messages.jsonl` or `collection/{session_id}/messages.jsonl`):
```json
{
  "timestamp": "ISO-8601",
  "user_message": "...",
  "deliberation_log": [
    { "member_name": "...", "position": 1, "response": "...", "escalation_triggered": false }
  ],
  "final_response": "..."
}
```

Conversation context schema (`chat/context.json`):
```json
{
  "user_id": "...",
  "turn_count": 0,
  "last_updated": "ISO-8601",
  "summary": ""
}
```

Collection session schemas:
- `session.json`: `{"session_id": str, "user_id": str, "plan_id": str, "status": "active"|"complete", "created_at": ISO-8601, "completed_at": ISO-8601|null}`
- `collected.json`: `{"field_name": "value", ...}`
- `context.json`: `{"session_id": str, "current_field": str, "fields_remaining": list[str], "last_updated": ISO-8601}`

Exact implementations to use (from `data-engineer.md`):

```python
def user_dir(self, user_id: str) -> Path:
    if len(user_id) < 4:
        raise ValueError(f"user_id must be at least 4 characters, got: {user_id!r}")
    return self.base / "users" / user_id[0:2] / user_id[2:4] / user_id

def append_jsonl(self, path: Path, record: dict[str, Any]) -> None:
    path.parent.mkdir(parents=True, exist_ok=True)
    with open(path, "a") as f:
        fcntl.flock(f, fcntl.LOCK_EX)
        try:
            f.write(json.dumps(record) + "\n")
            f.flush()
            os.fsync(f.fileno())
        finally:
            fcntl.flock(f, fcntl.LOCK_UN)

def write_json(self, path: Path, data: dict[str, Any]) -> None:
    path.parent.mkdir(parents=True, exist_ok=True)
    tmp = path.with_suffix(".tmp")
    with open(tmp, "w") as f:
        fcntl.flock(f, fcntl.LOCK_EX)
        try:
            json.dump(data, f, indent=2)
            f.flush()
            os.fsync(f.fileno())
        finally:
            fcntl.flock(f, fcntl.LOCK_UN)
    tmp.replace(path)  # atomic rename

def read_json(self, path: Path) -> dict[str, Any]:
    if not path.exists():
        return {}
    with open(path, "r") as f:
        fcntl.flock(f, fcntl.LOCK_SH)
        try:
            return json.load(f)  # type: ignore[no-any-return]
        finally:
            fcntl.flock(f, fcntl.LOCK_UN)

def read_jsonl(self, path: Path) -> list[dict[str, Any]]:
    if not path.exists():
        return []
    records: list[dict[str, Any]] = []
    with open(path, "r") as f:
        fcntl.flock(f, fcntl.LOCK_SH)
        try:
            for line in f:
                line = line.strip()
                if line:
                    records.append(json.loads(line))
        finally:
            fcntl.flock(f, fcntl.LOCK_UN)
    return records
```
