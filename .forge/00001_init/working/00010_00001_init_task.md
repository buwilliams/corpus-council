# Task 00010: Implement deliberation.py — council deliberation engine with escalation handling

## Role
programmer

## Objective
Implement `src/corpus_council/core/deliberation.py` with `run_deliberation(user_message: str, corpus_chunks: list[ChunkResult], members: list[CouncilMember], llm: LLMClient) -> DeliberationResult`. The engine iterates members from highest `position` to lowest, calls each member's LLM prompt (rendered from `member_deliberation.md`), checks the escalation rule (rendered from `escalation_check.md`), and — if escalation is triggered — skips remaining members and passes escalation context to position-1 for resolution via `escalation_resolution.md`. Position-1 always runs last and always produces the final response, either via `final_synthesis.md` (normal path) or `escalation_resolution.md`. Must pass `mypy --strict` and `ruff check`.

## Context
Task 00005 created `ChunkResult` in `src/corpus_council/core/retrieval.py`. Task 00006 created `CouncilMember` in `src/corpus_council/core/council.py`. Task 00007 created `LLMClient` in `src/corpus_council/core/llm.py`. Task 00008 created the prompt templates.

Deliberation pipeline:
1. Sort members by `position` descending (highest position first = least authoritative first). Position-1 is last.
2. For each member (excluding position-1):
   a. Render `member_deliberation.md` with: `member_name`, `persona`, `primary_lens`, `role_type`, `user_message`, `corpus_chunks` (formatted as text), `prior_responses` (formatted log of prior member outputs).
   b. Call `llm.call("member_deliberation", context)` to get the member's response.
   c. Render `escalation_check.md` with: `escalation_rule`, `member_response`.
   d. Call `llm.call("escalation_check", escalation_context)` to check if escalation triggered.
   e. If response starts with `TRIGGERED`: set `escalation_triggered = True`, record the member and reason, break out of the loop.
3. For position-1 (always last):
   a. If escalation was triggered: call `llm.call("escalation_resolution", {...})` with escalation context.
   b. If no escalation: call `llm.call("final_synthesis", {...})`.
   c. The result is the `final_response`.
4. Return `DeliberationResult`.

Helper for formatting corpus chunks for template context:
```python
def _format_chunks(chunks: list[ChunkResult]) -> str:
    if not chunks:
        return "No relevant corpus context available."
    return "\n\n---\n\n".join(
        f"[Source: {c.source_file}, chunk {c.chunk_index}]\n{c.text}"
        for c in chunks
    )
```

Helper for formatting prior responses:
```python
def _format_prior_responses(log: list[MemberLog]) -> str:
    if not log:
        return "No prior responses."
    return "\n\n".join(
        f"**{entry.member_name} (position {entry.position}):**\n{entry.response}"
        for entry in log
    )
```

No inline prompt strings. All LLM calls go through `llm.call(template_name, context)`.
