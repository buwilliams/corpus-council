# Task 00011: Implement conversation.py — conversation mode orchestration

## Role
programmer

## Objective
Implement `src/corpus_council/core/conversation.py` with `run_conversation(user_id: str, message: str, config: AppConfig, store: FileStore, llm: LLMClient) -> ConversationResult`. This function loads prior conversation context from `chat/context.json`, retrieves corpus chunks, runs full council deliberation, and persists the full turn to `chat/messages.jsonl` and `chat/context.json`. Resumable: if `context.json` exists for `user_id`, prior state is loaded. Must pass `mypy --strict` and `ruff check`.

## Context
Task 00002 created `AppConfig`. Task 00003 created `FileStore` (with `chat_messages_path`, `chat_context_path`). Task 00005 created `retrieve_chunks`. Task 00006 created `load_council`. Task 00007 created `LLMClient`. Task 00010 created `run_deliberation` and `DeliberationResult`.

`ConversationResult` dataclass:
```python
@dataclass
class ConversationResult:
    user_id: str
    response: str
    turn_count: int
```

Conversation context loaded from / saved to `chat/context.json`:
```json
{
  "user_id": "...",
  "turn_count": 0,
  "last_updated": "ISO-8601",
  "summary": ""
}
```

Full turn persisted to `chat/messages.jsonl` (one JSON object per line):
```json
{
  "timestamp": "ISO-8601",
  "user_message": "...",
  "deliberation_log": [
    { "member_name": "...", "position": 1, "response": "...", "escalation_triggered": false }
  ],
  "final_response": "..."
}
```

Pipeline for `run_conversation`:
1. Load context: `store.read_json(store.chat_context_path(user_id))` → `context: dict`. If empty dict, initialize with `{"user_id": user_id, "turn_count": 0, "last_updated": "", "summary": ""}`.
2. Retrieve corpus chunks: `retrieve_chunks(message, config)`.
3. Load council: `load_council(config)`.
4. Run deliberation: `run_deliberation(message, chunks, members, llm)`.
5. Increment turn count.
6. Persist turn to `messages.jsonl`: use `store.append_jsonl(store.chat_messages_path(user_id), turn_record)`.
7. Persist updated context to `context.json`: `store.write_json(store.chat_context_path(user_id), updated_context)`.
8. Return `ConversationResult`.

The `turn_record` dict to append must include `timestamp` (ISO-8601 from `datetime.now(timezone.utc).isoformat()`), `user_message`, `deliberation_log` (serialized from `DeliberationResult.deliberation_log`), and `final_response`.

Use `from __future__ import annotations`. Import `datetime`, `timezone` from standard library.

No inline LLM prompts. No direct `open()` calls — use `store` for all user data I/O.
