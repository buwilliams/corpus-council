# Task 00004: Implement corpus.py — flat-file corpus ingestion and chunking

## Role
programmer

## Objective
Implement `src/corpus_council/core/corpus.py` with the `ingest_corpus(config: AppConfig) -> IngestResult` function that reads `.md` and `.txt` files from `config.corpus_dir`, splits them into text chunks of at most `config.chunk_max_size` characters, and persists each chunk as a flat JSON file under `data/chunks/{source_hash}/{chunk_index}.json`. Idempotent: skip existing chunks based on `source_hash` + `chunk_index`. Return a typed `IngestResult` with `chunks_created: int` and `files_processed: int`.

## Context
Task 00001 created the scaffold. Task 00002 created `AppConfig` in `src/corpus_council/core/config.py`. Task 00003 created `FileStore` in `src/corpus_council/core/store.py`.

Corpus chunk metadata stored at `data/chunks/{source_hash}/{chunk_index}.json`:
```json
{
  "chunk_id": "uuid4-string",
  "source_file": "relative/path/to/file.md",
  "source_hash": "sha256-hex-of-file-content",
  "chunk_index": 0,
  "text": "chunk content here",
  "char_start": 0,
  "char_end": 512
}
```

Chunking algorithm:
- Read file content as UTF-8 text
- Split into non-overlapping segments of at most `chunk_max_size` characters
- Try to split on paragraph boundaries (`\n\n`) when possible; fall back to hard split at `chunk_max_size`
- Each segment becomes one chunk
- `char_start` and `char_end` are character offsets in the original file content

Idempotency:
- Compute `source_hash = hashlib.sha256(content.encode()).hexdigest()`
- Check if `data/chunks/{source_hash}/` directory already exists with the expected chunk files
- If all chunk files already exist, skip this file (count it in `files_processed` but not in `chunks_created`)

The chunks directory is at `config.data_dir / "chunks"`, not inside `data/users/`. This is corpus data, not user data — do NOT use `FileStore` for chunk metadata. Use direct file I/O with `pathlib.Path`.

`IngestResult` is a `dataclass`:
```python
@dataclass
class IngestResult:
    files_processed: int
    chunks_created: int
```

Only `.md` and `.txt` files are processed. All other extensions are silently ignored.

Use `from __future__ import annotations`.
