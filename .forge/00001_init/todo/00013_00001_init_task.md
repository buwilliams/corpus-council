# Task 00013: Implement FastAPI Pydantic models in api/models.py

## Role
api-designer

## Objective
Create `src/corpus_council/api/models.py` with all Pydantic request and response models for every FastAPI endpoint. All models use `snake_case` field names, `model_config = ConfigDict(extra="forbid")`, and are fully typed. Must pass `mypy --strict` and `ruff check`.

## Context
Task 00001 created the scaffold including `src/corpus_council/api/__init__.py`. Task 00012 created `CollectionSession`. The API has six endpoints with these contracts (from `api-designer.md`):

`POST /conversation`:
- Request: `ConversationRequest` — `user_id: str`, `message: str`
- Response 200: `ConversationResponse` — `response: str`, `user_id: str`
- Response 500: `ErrorResponse` — `error: str`

`POST /collection/start`:
- Request: `CollectionStartRequest` — `user_id: str`, `plan_id: str`
- Response 201: `CollectionStartResponse` — `user_id: str`, `session_id: str`, `first_prompt: str`
- Response 404: `ErrorResponse`

`POST /collection/respond`:
- Request: `CollectionRespondRequest` — `user_id: str`, `session_id: str`, `message: str`
- Response 200: `CollectionRespondResponse` — `user_id: str`, `session_id: str`, `prompt: str | None`, `status: Literal["active", "complete"]`, `collected: dict[str, Any]`
- Response 404: `ErrorResponse`

`GET /collection/{user_id}/{session_id}`:
- Response 200: `CollectionStatusResponse` — `user_id: str`, `session_id: str`, `status: str`, `collected: dict[str, Any]`, `created_at: str`
- Response 404: `ErrorResponse`

`POST /corpus/ingest`:
- Request: `CorpusIngestRequest` — `path: str`
- Response 200: `CorpusIngestResponse` — `chunks_created: int`, `files_processed: int`
- Response 422: FastAPI default

`POST /corpus/embed`:
- Request: no body (use `None` or an empty model)
- Response 200: `CorpusEmbedResponse` — `vectors_created: int`

`ErrorResponse` is shared across all error cases: `error: str`.

All models must use `from pydantic import BaseModel, ConfigDict` and set `model_config = ConfigDict(extra="forbid")`.
Use `from __future__ import annotations` and `from typing import Any, Literal`.

## Steps
1. Create `src/corpus_council/api/models.py` with `from __future__ import annotations`.
2. Import `BaseModel`, `ConfigDict` from pydantic; `Any`, `Literal` from typing.
3. Define `ErrorResponse` with `error: str`.
4. Define all request models: `ConversationRequest`, `CollectionStartRequest`, `CollectionRespondRequest`, `CorpusIngestRequest`.
5. Define all response models: `ConversationResponse`, `CollectionStartResponse`, `CollectionRespondResponse`, `CollectionStatusResponse`, `CorpusIngestResponse`, `CorpusEmbedResponse`.
6. All models set `model_config = ConfigDict(extra="forbid")`.
7. Run `uv run mypy src/corpus_council/api/models.py` (via `uv run mypy src/corpus_council/core/`) — note mypy is configured strict on `core/` only; ensure models.py has no type errors visible from core.
8. Run `uv run ruff check src/` and `uv run ruff format --check src/`.
