# Council

## Roles

- **programmer** — Implements Python code across all core modules, API, and CLI
- **tester** — Writes and validates the test suite; ensures coverage and real-implementation requirements are met
- **product-manager** — Ensures deliverables align with the product spec and constitution; guards against scope creep
- **api-designer** — Owns the FastAPI endpoint contracts, request/response shapes, and CLI interface design
- **data-engineer** — Owns the flat-file store design, sharding strategy, fcntl locking, JSONL/JSON schemas, and ChromaDB integration
- **security-engineer** — Reviews API key handling, file I/O safety, and input validation at system boundaries
